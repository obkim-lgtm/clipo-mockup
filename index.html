개별/일괄 작성 역할 구분
■ 개별 작성 — 한 학생을 깊게 처리
파일 업로드 → AI 특성 추출 → 특성 확인·보완 → AI 초안 생성 → 교사 최종 수정
■ 일괄 작성 — 준비된 특성으로 여러 명을 빠르게 완료
특성 확인 → AI 일괄 생성 → 교사 최종 기록 동시 확정
[개별 작성 or 활동지 일괄 업로드] → 특성 입력 완료 → [일괄 작성] → 완료
