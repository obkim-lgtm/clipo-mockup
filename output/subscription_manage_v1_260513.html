사용 방법

클리포의 AI 기능을 사용하면 학생 1명당 크레딧 1개가 차감됩니다. 여러 종류의 크레딧이 있으면 아래 순서대로 차감돼요.

  1. 데일리 크레딧 — 매일 충전되어 그날 안 쓰면 사라지니 먼저 사용
  2. 쿠폰·온보딩 크레딧 — 무료로 받은 종량제부터
  3. 보너스 크레딧 — 이용권 구매 시 받은 것
  4. 학교 공유 크레딧 — 학교에서 받은 것
  5. 추가 충전 크레딧 — 직접 구매한 것 (가장 마지막)

같은 순위 안에서는 만료일이 가까운 것부터 차감돼요.

데일리 크레딧

매일 자정에 새로 충전되며, 그날 안에 사용하지 않으면 사라져요.

  • 이용권 쿠폰 : 쿠폰 만료일까지 매일 충전
종량제 크레딧

사용하지 않아도 잔량이 그대로 남아요. 아래 유효 기간이 지나면 소멸됩니다.

  • 추가 충전 크레딧 : 구매일로부터 1년 (이용권이 만료되어도 사용 가능)
  • 학교 공유 크레딧 : 학교 이용권 만료일까지
  • 보너스 크레딧 : 함께 받은 이용권 만료일까지
  • 온보딩 크레딧 : 지급일로부터 6개월
  • 쿠폰 크레딧 : 쿠폰별 설정 기간
